# Credit Scoring Model

## Project Overview
This project predicts creditworthiness using Machine Learning. A Logistic Regression model is trained on the German Credit Dataset to classify customers based on their credit risk.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy

## Dataset
German Credit Dataset

## Steps Performed
1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. Logistic Regression Model Training
6. Prediction and Evaluation

## Result
Model Accuracy: 77.5%
