# Credit Scoring Model

## Project Overview

This project is a Machine Learning-based Credit Scoring Model developed using Python. The model predicts the creditworthiness of customers using the German Credit Dataset. Logistic Regression is used as the classification algorithm to determine whether a customer is a good or bad credit risk.

## Objectives

* Predict customer creditworthiness.
* Build a classification model using Logistic Regression.
* Evaluate model performance using multiple metrics.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Dataset

The project uses the German Credit Dataset, which contains customer financial information and credit risk details.

## Data Preprocessing

* Loaded the dataset using Pandas.
* Converted categorical values into numerical values using Label Encoding.
* Split the dataset into training and testing sets.

## Machine Learning Model

* Algorithm: Logistic Regression
* Train-Test Split: 80% Training, 20% Testing

## Evaluation Metrics

The model performance was evaluated using:

* Accuracy Score
* Precision
* Recall
* F1-Score
* ROC-AUC Score
* Confusion Matrix

## Results

The model successfully predicts customer creditworthiness and achieved good classification performance on the test dataset.
